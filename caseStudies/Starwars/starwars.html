May the Force be with you!
The Star Wars program was for a class that I took in my sophomore year. This website goes in detail on the different films in the industry listing them from the oldest to the newest. Along with this page, I also created a page that displays all the star war ships that are seen in the films. When hovering over the name of the ship, the image hovers over the right and in the background are blue and red later beams that go from side to side.
Project Details
There was no problem statement for this project seeing as it was more of a learning opportunity. While doing this project I had a chance to learn JavaScript and how to tie it all with HTML and CSS. And the user audience for it were just my peers who would critique me on my code and design of the whole website. Another audience member could potentially be a star wars fanatic who could stumble upon the page and have a fun time interacting and viewing the different ships.
As far as roles and responsibilities go, this was all coded and designed by me. I had a great time learning how to take information from an array and present it on my site where I can style it however I please. I also learned a few styles techniques in CSS and how can incorporate it all to a simple HTML page.
The end result turned out to be an amazing site that I could add to my portfolio and showcase my coding skills and design skills. This project made me fall in love with JavaScript and I have been using it ever since.
